A [streamlit](https://streamlit.io/) application created as part of the requirements for the [Snowflake hands-on workshop](https://learn.snowflake.com/en/courses/uni-ess-dabw/).
The workshop helped develop skills in building data applications using Snowflake and Streamlit.

The client app can be viewed using the public url https://melaniessmoothies-jklol.streamlit.app/ but the database updates may not work since this uses a trial Snowflake account.
